# Profile README Quickstart

This is the shortest path from clone to a first successful local verification pass.

## Requirements

- `git`
- `bash`
- `python3`

## 1. Clone

```bash
git clone https://github.com/inaciovasquez2020/inaciovasquez2020.git
cd inaciovasquez2020
```

## 2. Check tools

```bash
python3 --version
git --version
```

## 3. Run local checks

```bash
python3 -m pytest -q
```

## 4. Review the main routing surfaces

- `README.md`
- `docs/`
- `artifacts/`
- `tests/`
